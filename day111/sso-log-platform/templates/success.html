About This Project

This is a Single Sign-On (SSO) Log Platform that demonstrates secure authentication using enterprise identity providers like Google and Okta. The platform provides:

  • Secure OAuth 2.0 authentication flow
  • Multi-tenant architecture support
  • JWT-based session management
  • Redis-backed state management
  • Enterprise-grade security headers
